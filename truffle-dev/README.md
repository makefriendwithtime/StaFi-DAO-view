# vue-box

#### 介绍

基于truffle文档的truffle-boxes的webpack盒子，本人做了修改，做成vue的使用案例盒子。

注意：
不要使用Ganache，会出现莫名其妙的错误。
#### 软件架构
软件架构说明
truffle 
webpack
vue

#### 安装教程

1.  cd vuebox
2. npm install

#### 使用说明
首先保证在vue-box目录下
1.  truffle devlop
2.  compile
3.  migrate
4.  cd vuebox
5.  npm run serve
